核心流程
觀察／閱讀 → 提問／分析 → 詮釋／推理 → 歸納／整理 → 遷移／連結 → 表達／反思
能力白話意思對應作業
觀察／閱讀仔細看、仔細讀，不放過細節空間觀察、散赤人、文本分析
提問／分析問「為什麼」、拆解作者手法空間觀察、散赤人、論證分析
詮釋／推理讀出言外之意、找出背後原理散赤人、歌曲分析、生命起源
歸納／整理提煉成通則、畫成圖心智圖、空間觀察進階題
遷移／連結用到別領域、連到自身經驗空間觀察、ORID、歌曲分析
表達／反思寫出來、講出來、提出看法口頭報告、ORID、各反思題
一句話記住全部
不管讀什麼、看什麼，都用同一套流程：看清楚 → 問為什麼 → 想出道理 → 整理成圖／話 → 連到別處 → 說出我的看法。老師最看重「為什麼」和「我的看法」這兩段。

Shuchen Wang 老師 · 英文模組「夢想實踐家」

教四種閱讀／反思工具，課文是 Dream Big, Start Small
1ORID 反思法 影片／經驗反思▸

在教什麼：一套「想事情的固定順序」，看完影片或經歷一件事後照四步走就有話可說。

步驟問自己範例（回音法影片）
O 客觀我看到／聽到什麼事實？講者說發音要用回音法並示範單字。
R 感受我有什麼感覺？哪裡驚訝？很驚訝，原來連阿滴都會唸錯。
I 詮釋這對我有什麼意義／領悟？唸錯不丟臉，以後不用怕開口。
D 決定接下來要採取什麼行動？決定每天用回音法練 3 個單字。
怎麼拿高分
四步都要寫滿，I（詮釋）和 D（決定）是評分重點。很多人只寫到「很有趣」就停（那只到 R），一定要寫出「為什麼有感」和「我要怎麼做」。
2The Echo Method 回音法 發音實作▸

在教什麼：像山谷回音，聽一句、立刻模仿一句，模仿語調、節奏、重音。

要交什麼：抄負責的逐字稿段落、用紅筆切斷句子、記錄練習次數、寫實踐體悟。

怎麼拿高分
① 練習次數誠實且夠多（寫 5～10 次比 1 次有說服力）；② 體悟具體——「我發現 th 的咬舌位置」比「我覺得有進步」好太多。
3摘要 Summary 抓重點▸

在教什麼：用自己的話抓骨架。Dream Big 三條黃金法則＝①從小改變 ②訂具體計畫 ③一次專注一個目標。

摘要填空正解（已對照原文）
1=dreams／2=small／3=unrealistic／4=difficult／5=make specific plans／6=focus／7=golden（three golden rules）
4文章結構分析 看作者鋪陳▸

在教什麼：看出說明文套路。gaming disorder 排序正解：

(d) 定義 → (b) 症狀 → (c) 治療 → (a) 呼籲行動

學會套路，讀任何說明文都能預測下一段，讀得又快又懂。

林世奇 老師 · 國文〈散赤人〉讀後

深度閱讀 × 文學鑑賞，像評論者一樣分析作品

八個問題各對應一個分析角度。下面整理每題在考什麼、答題方向。

題分析工具白話＋答題方向
一人物描寫手法找作者用的手法（動作、語言、細節、對比）。例：阿公喊「秤了沒？」一句寫活吝嗇。
二扁平 vs 圓形人物嬸婆是扁平人物——性格單一、全篇沒變化、可一句話概括。
三敘事技巧用小故事＋閩南語俗諺＋引經典推動情節，畫面感強。
四主題思辨貧窮影響：尊嚴受損、人際緊張、奪食。作者認為「人窮志短」才是常態。
五中心意旨判讀作者沒明講，要讀出：質疑「寫歷史傳記的人能看見多少真相」。
六幽默 vs 滑稽幽默＝局外人輕鬆旁觀；滑稽＝身在其中，又笑又痛。本文是滑稽。
七意象與鄉土文學本文反美化鄉土：故意拆穿「民風淳樸」的浪漫想像。
八個人讀後感整合以上，提出自己的觀點與經驗連結。
兩把解讀鑰匙（抓到就高分）
鑰匙一・滑稽不是幽默：阿嬤詛咒嬸婆、嬸婆卻踮腳偷看——你會笑，笑完心裡發酸，這就是「滑稽」。
鑰匙二・反美化書寫：文末作者明說學者把洲尾寫成「民風淳樸」，但真相是貧窮讓人志短。真正想說的是「真相往往被美化的文字掩蓋」。
怎麼拿高分
國文開放題一定要引用文中具體段落當證據，再加上「我的感受／類似經驗」。只寫感想＝低分；有引文＋個人連結＝高分。

鄭榮輝 老師 · 心智圖 ＋ 文本分析

視覺化整理 ＋ 論證閱讀，共四份作業

🗂️ 心智圖系列（視覺化整理）

1學科知識心智圖 紙本拍照 · 5/3▸

選一個學科主題手繪心智圖、拍照上傳。結構：中心主題 → 主分支（大概念）→ 次分支（細節）。

2個人興趣心智圖 Xmind PDF · 5/3▸

用 Xmind 軟體做個人興趣心智圖，輸出 PDF 上傳。

心智圖怎麼拿高分（通用）
① 分層清楚至少三層；② 用關鍵詞不寫整句，一節點一詞；③ 同層概念份量相當；④ 用顏色／圖示分類。評分看「結構清晰」不是「畫得漂亮」。

🔬 文本分析系列（論證閱讀）

3生命起源論證文本分析 5/10▸

核心問法：作者主張什麼？用了哪些證據？推論是否合理？有沒有其他可能解釋？

4細胞膜模式建模文本分析 5/10▸

核心問法：科學家怎麼從實驗證據建立「細胞膜模型」？模型怎麼隨新證據被修正？

論證分析萬用四步框架
1. 主張 Claim＝作者最後要說什麼？
2. 證據 Evidence＝拿什麼資料／實驗支持？
3. 推理 Reasoning＝證據怎麼連到主張？
4. 反思 Limitation＝有沒有漏洞、其他解釋、未解問題？
怎麼拿高分
多數人只答到「主張」，分數在第 3、4 點——能說出「證據如何推導結論」「模型有何限制／曾被修正」，才是這份作業要的深度。

吳宜玲 老師 · 年代歌曲文本分析

文本詮釋＋時代連結，小組書面＋口頭報告

把流行歌當「文本」，從歌詞讀出社會、政治、價值觀。

作業要求
選歌：一首能代表台灣 2010～2025 時代意義的歌（以華語歌為主）。
檔名：「班級_組別_歌名」，例：一信_第一組_玫瑰少年。
書面（三頁）：
  1. 挑選此代表歌曲的原因、歷程
  2. 投影片插入歌曲影音連結，寫歌名／創作者／歌手
  3. 列出歌詞關鍵句至少 3 句，說明為何代表這時代（社會、政治、經濟、愛情觀、價值觀等）
首頁註明主題、班級、小組成員座號姓名。
口頭報告扣分地雷（老師特別列出）
  • 不可盯著螢幕念稿，要消化後用自己的話說
  • 眼神必須看台下同學
  • 時間不可少於 1 分鐘、不可大於 3 分鐘
  • 介紹關鍵歌詞時必須解說該句意義
  • 歌曲放一遍就好，副歌結束即可停止
怎麼拿高分
選歌就贏一半——挑明顯反映社會議題的歌（〈玫瑰少年〉談性別平權、〈島嶼天光〉談社運），關鍵句要寫「這句反映什麼社會現象／價值觀」，別只翻譯字面。

空間觀察與分析（Part1）

探究式思考：把校園當研究對象

訓練「觀察→提問→推理→歸納→遷移→批判」。（你這份已完成且寫得很完整。）

階段在做什麼對應題目
觀察注意校園空間怎麼安排全題前提
提問對理所當然的現象問為什麼1～6 題
推理找出原理（動靜分離、噪音隔離、物流安全）每題「●」分析
歸納把零散觀察提煉成通則進階題一
遷移放大到都市、國土計畫進階題三
批判找缺點＋提改進方案進階題二
怎麼拿高分
每個答案要有「原則名稱＋具體理由」兩層。不要只寫「比較安靜」，要寫「動靜分離：遠離操場，減少體育課噪音干擾備考」。你目前的寫法正是滿分寫法。

所有「分析／反思題」的高分公式
原則／概念名稱　＋　具體證據（引文／例子）　＋　我的詮釋或看法
